# 📦 Küçük İşletmeler İçin Sipariş ve Stok Karar Destek Sistemi

🌍 **Canlı Web Sitesi (Demo):** [Buraya Tıklayarak Arayüzü İnceleyebilirsiniz](https://small-business-decision-support-system.github.io/order-stock-dss/).

## 📌 Proje Hakkında
Bu proje, küçük işletmelerin envanter yönetimini optimize etmek amacıyla geliştirilmiş çok disiplinli bir **Karar Destek Sistemidir (DSS)**. Sistem, deterministik modeller yerine **%20 talep değişkenliğini baz alan Stokastik Model** kullanılarak tasarlanmıştır.

Arayüz üzerinden ürünlerin anlık stok durumları, Emniyet Stoğu (SS), Yeniden Sipariş Noktası (ROP) ve Ekonomik Sipariş Miktarı (EOQ) gibi kritik metrikler takip edilebilir.

## 🛠️ Kullanılan Teknolojiler
* **Frontend:** React.js (Vite)
* **Backend:** Python, FastAPI
* **Veritabanı:** PostgreSQL (Render üzerinde barındırılmaktadır)
* **Analitik:** Python (Pandas, NumPy) ile ABC Analizi ve Stokastik Hesaplamalar
* **Donanım:** Tinkercad / Proteus (Sensör simülasyonları)

## 📂 Klasör İçeriği ve Düzenleme Kuralları
Ekip üyelerinin (Yaren ve Mustafa) aşağıdaki düzeni takip etmesi rica olunur:

1.  **Reports/**: Hazırlanan Word/PDF raporları (Örn: `envanter_raporu.docx`).

3.  **Models/**: Python ile yapılan matematiksel hesaplama scriptleri.
